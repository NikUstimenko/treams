treams.TMatrix.globalmat¶

TMatrix.globalmat(basis=None)¶

Global T-matrix.

Calculate the global T-matrix starting from a local one. This changes the T-matrix.

Parameters:

  • origin (array, optional) – The origin of the new T-matrix

  • modes (array, optional) – The modes that are considered for the global T-matrix

  • interacted (bool, optional) – If set to False the interaction is calculated first.

Returns

TMatrix
