ptsa.SMatrix.add¶

SMatrix.add(sm)¶

Couple another S-matrix on top of the current one

See also ptsa.SMatrix.stack() for a function that does not change the current S-matrix but creates a new one.

Parameters:

  • items (SMatrix) – S-matrices in their intended order

  • check_materials (bool, optional) – Check for matching material parameters at each S-matrix

  • check_materials – Check for matching modes at each S-matrix

Returns:

SMatrix
