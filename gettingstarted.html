Getting started¶

Installation¶

Installation using conda¶

Currently treams is not available as a conda package. A local repository can be installed in development mode (see Setting up the development environment).

Installation using pip¶

To install the package with pip, use

pip install git+https://git.scc.kit.edu/photonics/treams.git

How to use treams¶

Where to go from here¶

A couple of simulations with varying degree of complexity can be found in Examples. There’s also a complete Reference of the API. If you want to get contribute to the development of the program, you can find relevant information under Development and contributing.
